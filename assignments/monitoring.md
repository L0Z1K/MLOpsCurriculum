# Monitoring

- 각 Endpoint 마다 응답 시간의 평균/max/min 값을 확인한다.
- DB의 걸리는 부하 (트랜잭션 수, 평균/max/min 응답시간)
- DB 데이터 크기
- 에러가 발생한 경우, 어떤 input에 의해 발생한 것인지 확인할 수 있게.
